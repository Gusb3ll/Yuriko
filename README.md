# Yuriko

## Setup

Install dependencies:

```bash
yarn install
```

## Development

```bash
yarn dev
```

Client only:

```bash
yarn workspace client dev
```

Server only:

```bash
yarn workspace server dev
```

## Production

```bash
yarn build

yarn start
```

Client only:

```bash
yarn workspace client build

yarn workspace client start
```

Server only:

```bash
yarn workspace server build

yarn workspace server start
```
